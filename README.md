# لعبة الدمج الاحترافية 6x6 + DA

لعبة دمج الفواكه والخضار مع نظام DA وشحن الجواهر.

## الميزات
- يحصل المستخدم المسجل على 100 جوهرة مجانية عند التسجيل.
- تحويل كل 1000 نقطة إلى 10 DA.
- الزوار يمكنهم اللعب بدون تسجيل.
- يمكن للمستخدمين شراء جوهرة حسب رغبتهم، ويتم الإشعار عند الشحن.
- تخزين البيانات محليًا ومرن عبر `database.json`.

## تشغيل المشروع
1. تأكد من وجود Node.js مثبت.
2. افتح الطرفية في مجلد المشروع.
3. ثبّت الحزم:
